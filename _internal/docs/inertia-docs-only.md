# InertiaJS - Docs-Only Dependency

`inertiajs/inertia-laravel` (v3.x) was installed on 2026-04-08 purely so Laravel Boost can index its documentation for research purposes. It is **not configured or used** in the application.

## Safe to Remove

This package can be removed at any time with:

```bash
composer remove inertiajs/inertia-laravel
```

No middleware, routes, or views depend on it.
